Rashtrasant Tukadoji Maharaj Nagpur University (RTMNU), formerly Nagpur University, is a public state university located in Nagpur, Maharashtra. It is one of India's oldest universities, as well as the second oldest in Maharashtra. It is named after Rashtrasant Tukdoji Maharaj, a spiritual leader, orator, and musician from Vidarbha. The university is a member of the Association of Indian Universities and the Association of Commonwealth Universities

About IICC RTMNU,Nagpur

The Inter Institutional Computer Centre (IICC) of Nagpur University is a research and development centre that provides computing services to the university. It offers various courses in computer science, software engineering, networking technologies, database management systems and other related fields. IICC also has its own dedicated server for hosting websites and applications developed by students or faculty members. In addition to this, it provides technical support for hardware maintenance as well as network security solutions.
